  아직까지 대부분의 건축물에서의 연기감지기는 주로 규칙 위주 설계(Rule-based Design)에 의해 설치되고 있으나 최근 성능 위주 설계(Performance-based Design)의 중요성이 강조되고 있어, 특정 건축물들에는 필수적으로 화재 시뮬레이션으로 수많은 화재 시나리오를 수행하는 과정을 거치게 되었습니다. 
   화재 시뮬레이션을 진행하는 것에 많은 시간과 전문 지식이 필요한 만큼 실무자에게는 주어진 시간 내에 충분한 시뮬레이션을 수행하고 방재 설비를 설계하는 것에 많은 어려움을 겪고 있습니다. 
  이에 추후 수많은 시나리오를 일차적으로 빠르게 검토할 수 있는 연기 분포 모델을 개발하기 위해, 특정 화원에서 발생한 화재로 확산된 연기가 건물 내 평면으로 확산되는 확률을 계산하는 모델을 찾고자 연구를 수행했습니다.
